# TP2 Strapi G18

## Requisitos previos
- Node.js 18+
- pnpm (requerido)

## Instalaci√≥n

1. Instalar pnpm globalmente:
```bash
npm install -g pnpm
```

2. Clonar el repositorio:
```bash
git clone https://github.com/enzomolina10/tp2Strapi-g18.git
cd tp2Strapi-g18
```

3. Instalar dependencias del servidor:
```bash
cd server
pnpm install
```

4. Ejecutar en desarrollo:
```bash
pnpm run develop
```
